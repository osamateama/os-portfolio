# Osama Reda Portfolio

Welcome to my personal developer portfolio built with **React**, **Tailwind CSS**, and **Motion** for animations. This project showcases my skills, services, and featured projects in backend and full-stack .NET development.

---

## 🖥️ Features

- **Responsive Navbar** with scroll animation
- **Hero Section** with typing effect
- **Skills Section** with animated progress bars
- **Services Section** showing professional offerings
- **Projects Section** with interactive cards and GitHub links
- **Contact Section** with email, phone, and WhatsApp links
- **Downloadable CV** button
- Fully **responsive** and optimized for desktop and mobile
- Smooth **scroll progress** indicator
